これは、このダウンロード可能な RELAB データベースから目的のスペクトルを見つけるための簡単なガイドラインです。

1. Sample_Catalogue と Spectra_Catalogue を、catalogues/ フォルダーの下にダウンロードします。

2. Sample_Catalogue で、目的のサンプルのサンプル ID を見つけます。 次に、Spectra_Catalogue でそのサンプル ID を検索して、スペクトル ファイル名を見つけます。 サンプル写真があればご用意も可能です。

3. data/ フォルダーに移動してファイルを見つけます。 data/ フォルダーは、3 文字の PI コードとサンプル ID の最初の 2 文字に従って構造化されています。

たとえば、目的のサンプルの Sample_Catalogue に「LS-CMP-012」が含まれている場合、
Spectra_Catalogue で C1LS12 などのファイル名を特定し、data/cmp/ls/ フォルダーに移動します。
c1ls12.asc、c1ls12.txt などを見つけます。

同じスペクトル データには、.asc 形式と .txt 形式の 2 つのファイルがあります。

<ASC形式>
   1行目：データ点数
   2 行目 - : 波長 (nm)、反射率、標準偏差 (利用可能な場合)
                 (スペースで区切ってください)
   最終行：ファイル名とコメント

<TXT形式>
   1行目：サンプルIDとファイル名
   2行目：データ列ヘッダー
   3 行目 - : 波長 (ミクロン)、反射率、標準偏差 (利用可能な場合)
                 (タブ区切り)

Sample_Catalogue の他の列については、Catalogue_README.html ファイルを参照してください。

あるいは、ここに配置されているすべてのカタログとデータを含む圧縮 zip ファイルをダウンロードすることもできます。
